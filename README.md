# PDFMerge
# 📄 PDF 快速合併工具使用手冊

這是一款基於 Python 開發的桌面工具，專門用於將兩個 PDF 檔案合併為一個。透過直覺的**拖放 (Drag & Drop)** 操作，您可以輕鬆決定檔案的前後順序。

---

## 🚀 使用步驟

### 1. 啟動程式
執行 `pdfmerge.exe` 視窗將出現在螢幕正中央。

### 2. 放入檔案
從資料夾中選取 PDF 檔案，直接拖曳至視窗區塊內：
- **左側區塊 (藍色)**：此處的檔案內容將排在合併後的**前半部分**。
- **右側區塊 (黃色)**：此處的檔案內容將接在第一個檔案**之後**。

> **提示**：若想更換檔案，直接拖入新的檔案即可覆蓋。

### 3. 執行合併
當兩個區塊都成功顯示檔名後，視窗下方會出現綠色的 **「開始合併檔案」** 按鈕。
1. 點擊按鈕。
2. 選擇存檔路徑並為新檔案命名。
3. 點擊「存檔」，完成合併。

---

## 🛠 故障排除 (Q&A)

**Q：為什麼拖入檔案後沒有反應？**
A：請確保您的檔案後綴名為 `.pdf`。程式會自動過濾非 PDF 格式的檔案。

**Q：合併按鈕沒有出現？**
A：合併按鈕僅在「左」與「右」兩區都有檔案時才會顯示。

**Q：合併失敗並顯示錯誤訊息？**
A：請檢查您的 PDF 是否被加密（設定了開啟密碼）或者該檔案目前正被其他程式（如 Adobe Acrobat 或 Chrome）開啟中。
